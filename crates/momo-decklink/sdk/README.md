# DeckLink SDK 15.3

This directory contains the Blackmagic DeckLink SDK headers and samples.
These files are proprietary and not committed to git.

## Download

1. Go to https://www.blackmagicdesign.com/developer
2. Download "Desktop Video 15.3 SDK"
3. Extract `Linux/include/*` into `include/`
4. Extract `Linux/Samples/*` into `samples/` (optional)

## Structure

```
sdk/
├── include/          # C++ headers (DeckLinkAPI.h, LinuxCOM.h, etc.)
├── samples/          # Official C++ samples (Capture, DeviceList, etc.)
├── examples/         # Cross-platform examples
└── Blackmagic DeckLink SDK.pdf
```
